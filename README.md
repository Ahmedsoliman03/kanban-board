# 📋 Kanban Board

A modern, streamlined Kanban board application built with **Next.js**, **Zustand**, and **Tailwind CSS**. Designed for simplicity and performance.

---

## 🚀 Features

- **Efficient Task Management**: Organize tasks into columns effortlessly.
- **State Management**: Powered by `Zustand` for fast and predictable state updates.
- **Modern UI**: Styled with `Tailwind CSS 4.0` and `Material UI` icons for a premium look.
- **Responsive Design**: Works seamlessly across all devices.

---

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/)
- **State Management**: [Zustand](https://zustand-demo.pmnd.rs/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Icons**: [Material UI Icons](https://mui.com/material-ui/material-icons/)
- **Notifications**: [react-hot-toast](https://react-hot-toast.com/)

---

## ⚡ Setup Instructions

Follow these steps to get the project running locally:

### 1. Installation
Install the project dependencies using npm:

```bash
npm install
```

### 2. Development Server
Run the development server:

```bash
npm run dev
```

### 3. Access the App
Open [http://localhost:3000](http://localhost:3000) in your browser to see the application in action.

---

## 📖 Available Scripts

- `npm run dev` - Starts the development server.
- `npm run build` - Builds the application for production.
- `npm run start` - Starts the production server.
- `npm run lint` - Runs ESLint to check for code quality.
